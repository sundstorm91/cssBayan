# 🎹 CSS Bayan — аккордеон на чистом CSS

Аккордеон (раскрывающийся список) с мемами про разработку. Реализован полностью на CSS — без единой строчки JS. Тренировка селекторов, псевдоклассов, переходов и адаптива.

## 🛠️ Tech Stack

- **HTML5** — семантическая разметка
- **CSS3** — flexbox, transition, hover-эффекты, медиазапросы

## 🧩 Функционал

- Раскрытие карточек при наведении (hover)
- Смена иконки с плюса на крестик
- Анимация изображений при наведении (scale)
- Адаптив под 3 размера экрана (мобильный, планшет, десктоп)
- Никакого JS — только CSS

## 🧠 Что дал этот проект

- Понимание работы **псевдоклассов** (`:hover`, `:active`)
- Умение строить **аккордеон без JS**
- Опыт адаптивной вёрстки с использованием медиазапросов
- Тренировка **селекторов и специфичности**
